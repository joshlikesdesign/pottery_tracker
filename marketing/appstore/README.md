# App Store screenshot templates

This folder contains editable, production-size screenshot layouts for both device families:

- iPhone: `1242x2688` (portrait)
- iPad: `2752x2064` (landscape)

## Files

- `index.html` - all templates and copy blocks
- `styles.css` - layout, typography, color themes, and device framing

## Quick edit flow

1. Open `marketing/appstore/index.html`.
2. Replace each `<img src="...">` with your screenshot path.
3. Update each headline and supporting line.
4. Open `index.html` in a browser to review composition.

## Suggested copy sequence

1. Value proposition (`See What Needs Attention Today`)
2. Workflow progress (`Track Every Piece Through Every Stage`)
3. Fast entry (`Add New Work in Seconds`)
4. iPad workspace (`Manage Your Studio on a Full-Size Canvas`)
5. Organization and planning (`Filter by Status, Glaze, and Date`)

## Notes

- Keep headline lines short (4-8 words) for best balance.
- The templates are intentionally full-resolution for export-ready composition.
- If you need additional slides, duplicate an existing `<article class="shot ...">` block and swap content.
